# Stopwatch TUI

Multiple stopwatch terminal UI built with [Textual](https://textual.textualize.io/).

## About

Study project following the [official Textual library tutorial](https://textual.textualize.io/tutorial/), with additional features:

- **Stop All**: Stop all stopwatches simultaneously
- **Start All**: Start all stopwatches at once
- **Reset All**: Reset all stopwatches at once

## Features

- Multiple independent stopwatches
- Individual controls (start, stop, reset)
- Global controls to manage all stopwatches
- Responsive terminal interface

## Usage

```bash
python stopwatch.py
```

## Requirements

- Python 3.7+
- Textual

```bash
pip install textual
```
